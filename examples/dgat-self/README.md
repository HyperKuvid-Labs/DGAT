# dgat-self — DGAT analyzing itself

This folder contains the output DGAT produced when pointed at its own source tree.

| File | Description |
|---|---|
| `dgat_blueprint.md` | Synthesised architectural overview of the DGAT project, generated bottom-up from individual file descriptions |
| `file_tree.json` | Full hierarchical file tree with LLM-generated descriptions for every tracked file |
| `dep_graph.json` | Dependency graph with nodes (files) and edges (import relationships), each annotated with a plain-English explanation |

All descriptions were generated by **Qwen3.5-2B** running locally via vLLM.

These files are committed as a reference snapshot so you can explore the output format without running a scan yourself. The root-level equivalents (`dgat_blueprint.md`, `file_tree.json`, `dep_graph.json`) are `.gitignore`d — only this `examples/` directory is tracked.
